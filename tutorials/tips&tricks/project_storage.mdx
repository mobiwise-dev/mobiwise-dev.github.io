# Where are my projects stored ?

Mobirise stores your projects in this folder :

- C:\Users\<Highlight color="#e43f3f">User-Name</Highlight>\AppData\Local\Mobirise.com\Mobirise\<Highlight color="#e43f3f">projects</Highlight>

## What does this "projects" folder contain ?

One sub-folder per project. Each project is saved in a folder named according to this convention: 'project-YYYY-MM-DD_HHMMSS'.

## Exemple de dossiers 'project-YYYY-MM-DD_HHMMSS'

```
project-YYYY-MM-DD_HHMMSS
├── assets
|  └── images
|     ├── image1
|     ├── image2
|     └── hashes.json
├── fonts
|  └── Jost
|     ├── 100.ttf
|     ├── 100italic.ttf
|     ├── 200.ttf
|     ├── 200italic.ttf
|     ├── etc...
|     ├── italic.ttf
|     ├── regular.ttf
|     └── style.css
├── history
|  ├── project-YYYY-MM-DD_HHMMSS.mobirise_history
|  ├── project-YYYY-MM-DD_HHMMSS.mobirise_history
|  └── project-YYYY-MM-DD_HHMMSS.mobirise_history
├── project.mobirise
└── screenshot.png
```

All your site data (site name, theme, HTML, CSS, JS...) are in the **project.mobirise** file.
The `.mobirise_history` files have the same structure as **project.mobirise**. They are copies of your projects at a specific time.

:::tip
**project.mobirise** and `.mobirise_history` files are simple JSON format files. You can open them with your favorite code editor software.
:::
