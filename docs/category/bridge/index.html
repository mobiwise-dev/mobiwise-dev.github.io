Bridge

Bridge functions of the 'mbrApp.Core' object.

📄️ Bridge.addRecent

La fonction permet d'ajouter un élément récent dans la liste de fichiers ou de pages visités récemment.

📄️ Bridge.bind

La fonction Bridge.bind permet de lier une fonction de rappel à un événement particulier.

📄️ Bridge.decode

La fonction Bridge.decode prend deux paramètres : a et b.

📄️ Bridge.dirList

La fonction permet de lister les fichiers et les répertoires dans un répertoire donné.

📄️ Bridge.encode

La fonction Bridge.encode prend deux paramètres : a et b.

📄️ Bridge.getInfoLocal

📄️ Bridge.getLocalJSON

La fonction Bridge.getLocalJson est utilisée pour charger un fichier JSON à partir de la mémoire locale de l'appareil de l'utilisateur.

📄️ Bridge.getSettings

Cette fonction récupère les paramètres de configuration de l'application en fonction de la clé de configuration spécifiée en paramètre.

📄️ Bridge.setSettings

Cette fonction est utilisée pour définir les paramètres de l'application. Elle prend trois arguments : a, b et c.
