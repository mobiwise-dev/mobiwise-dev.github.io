## Functions list

- Works with HTML and PHP Pages
- Define the website language
- Append code (global or particular page):
  - Before `<!DOCTYPE>`
  - Inside `<head>`
  - After `<body>`
  - Before `</body>`
- Remove all Mobirise backlinks
- Remove all Mobirisre references
- Beautify your html code :
  - Remove comments
  - Remove empty lines
  - Indent the code
  - Minimify the code
- Minimify your css
- Remove SmoothScroll.js if Bootstrap 5
- Remove meta http-equiv='X-UA-Compatible'
- Add an .htaccess file
- Copy-paste page between website
- Rebuild user blocks
- Replace `http://` links to site Pages with `https://`
